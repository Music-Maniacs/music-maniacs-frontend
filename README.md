# Versiones usadas

## Yarn v1.22.19

Instalar Yarn: https://classic.yarnpkg.com/lang/en/docs/install/#debian-stable

## Node v20.5.0

Instalar desde nvm (Node Version Manager) https://github.com/nvm-sh/nvm

## Npm v9.8.0

# Iniciar la aplicacion

Instalar dependencias:

```bash
yarn install
```

Correr aplicacion:

```bash
yarn start
```
